\begin{frame}
    \frametitle{Основные результаты работы}
    \small
    \begin{enumerate}
        \item \textbf{Математическая модель:} глобальный план $G=(\overline{\overline{X}},\overline{\overline{Y}},\overline{\overline{Z}})$, целевая функция $\mu(G)=\langle\omega,\varphi,\tau\rangle\to\min$, система из~9~ограничений
        \item \textbf{Численный метод:} генетический алгоритм с~операторами мутации, глобальной мутации и~парного отбора; событийная модель расписания
        \item \textbf{Комплекс программ:} реализация модели и~численных методов; подтверждён результатами вычислительных экспериментов
    \end{enumerate}
    \vspace{0.2cm}
    \textbf{Соответствие направлениям исследований паспорта специальности 1.2.2:}
    \begin{itemize}
        \item \textbf{п.\,2}~--- разработка и~тестирование эффективных вычислительных методов \emph{(результат~2)}
        \item \textbf{п.\,3}~--- реализация численных методов в~виде комплекса программ для~вычислительного эксперимента \emph{(результат~3)}
        \item \textbf{п.\,6}~--- разработка системы имитационного моделирования \emph{(событийная модель расписания)}
        \item \textbf{п.\,8}~--- комплексное исследование с~применением математического моделирования и~вычислительного эксперимента \emph{(результаты~1,\,3)}
        \item \textbf{п.\,9}~--- постановка и~проведение численных экспериментов \emph{(результат~3)}
    \end{itemize}
\end{frame}
\note{
    Результаты соответствуют пяти из девяти направлений исследований паспорта специальности 1.2.2:
    п.~2~--- разработка вычислительных методов (генетический алгоритм),
    п.~3~--- реализация в комплексе программ,
    п.~6~--- система имитационного моделирования (событийная модель),
    п.~8~--- комплексное исследование с математическим моделированием,
    п.~9~--- постановка и проведение численных экспериментов.
}

\begin{frame}
    \frametitle{Научная новизна}
    \begin{enumerate}
        \item Разработана математическая модель совместной оптимизации производственных планов и расписаний в~целлюлозно-бумажной промышленности, в~которой решение представлено в~виде глобального плана $G=(\overline{\overline{X}},\overline{\overline{Y}},\overline{\overline{Z}})$, объединяющего выпуск тамбуров, планы раскроя и~очередь производственных событий
        \item Предложена событийная модель формирования согласованного производственного расписания, учитывающая многомашинную структуру цепочки БДМ~$\to$~склад~$\to$~ПРС/БРС, динамику складских состояний $(\overline{T}, \overline{\overline{T}})$, доступность полуфабрикатов, сроки заказов и~технологические ограничения
        \item Разработан численный метод на~основе генетического алгоритма для поиска допустимых глобальных планов, в~котором изменение плана раскроя сопровождается согласованным пересчётом плана производства и~расписания с~учётом взаимосвязи стадий производства и~ограничений ресурсов
    \end{enumerate}
\end{frame}
\note{
    Научная новизна работы состоит в~трёх основных результатах.
    Во\=/первых, предложена интегрированная математическая модель, в~которой одновременно рассматриваются производственный план и~расписание.
    Во\=/вторых, разработана событийная модель согласования стадий БДМ, склада и~ПРС/БРС с~учётом динамики складских состояний и~сроков заказов.
    В\=/третьих, предложен генетический алгоритм, который ищет не~только раскройные планы, но~и~допустимые глобальные планы, согласованные с~производством и~временной структурой процесса.
}
