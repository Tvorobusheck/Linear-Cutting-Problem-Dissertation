\addcontentsline{toc}{chapter}{Словарь терминов}  % Добавляем его в оглавление

% В данном словаре собраны ключевые понятия и сокращения,
% используемые в работе. Каждая запись состоит из термина и его определения.

\textbf{БДМ (бумагоделательная машина)} : оборудование для производства бумажного полотна (тамбуров) из целлюлозной массы; характеризуется шириной выпускаемой продукции, скоростью производства и временем смены плотности.

\fixme{\textbf{Блокировка (deadlock)} : состояние имитационного прогона, в котором исполнение плана не может быть продолжено (например, из-за недоступности полуфабриката при накопившихся отклонениях); доля прогонов с блокировкой используется как показатель устойчивости плана.}

\textbf{БРС (бобино‑резательный станок)} : разновидность продольно‑резательного станка, предназначенная для нарезки небольших рулонов («малых» заказов) на основе мастер‑рулонов, полученных после первой стадии раскроя.

\textbf{ГА (генетический алгоритм)} : алгоритм оптимизации, основанный на принципах эволюции; использует популяцию решений, операции мутации и кроссинговера для поиска оптимального плана.

\textbf{Генетический поиск (Genetic Search)} : итеративный стохастический алгоритм оптимизации, основанный на принципах эволюции; использует популяцию решений, операции мутации и кроссинговера для поиска оптимального плана.

\textbf{Глобальная мутация (GlobalMutate)} : оператор генетического алгоритма, модифицирующий сразу все планы раскроя в глобальном плане и пересчитывающий соответствующие планы производства и расписание.

\textbf{Глобальный план} : комплексный план, включающий планы производства на БДМ, планы раскроя на ПРС/БРС и расписание, описывающее порядок операций во времени.

\textbf{Заказ} : требование на изготовление определённого количества рулонов заданной ширины, плотности, марки бумаги, количества слоёв и срока исполнения.

\textbf{ЗЛР (задача линейного раскроя)} : классическая задача комбинаторной оптимизации, состоящая в разрезании заготовок фиксированной длины (ширины) на части заданных размеров с минимизацией отходов; в настоящей работе рассматривается в контексте целлюлозно‑бумажного производства.

\fixme{\textbf{Имитационное моделирование (дискретно-событийное)} : метод оценки устойчивости глобального плана, при котором исполнение расписания воспроизводится как последовательность дискретных событий с учётом случайных отклонений длительностей операций и отказов оборудования; структура плана при этом не изменяется.}

\textbf{Кромка} : технологический отступ с обеих сторон тамбура, который необходимо срезать при раскрое; в модели обозначается $V$ и учитывается как отход.

\textbf{Кроссинговер (Crossingover)} : оператор генетического алгоритма, который из пары глобальных планов выбирает лучший по критериям план для перехода в следующую популяцию.

\textbf{Марка бумаги} : класс (бренд) бумаги, характеризующий состав и технологию производства; смена марки на БДМ требует дополнительного времени.

\textbf{Мутация (Mutation)} : генетическая операция, которая изменяет план раскроя путём случайной перестановки форматов и раскроев; направлена на поиск новых комбинаций.

\textbf{Операция} : элемент расписания, характеризующий начало и окончание производства тамбура или раскроя рулонов; содержит информацию о типе операции, используемом оборудовании и состоянии складов.

\fixme{\textbf{Отказ оборудования} : случайное событие в имитационной модели, прерывающее выполнение операции на случайное время ремонта; время до отказа распределено экспоненциально с интенсивностью $\nu^{\mathrm{пр}}$ или $\nu^{\mathrm{рез}}$, время ремонта --- экспоненциально со средним $\theta$; после ремонта операция продолжается с прерванного места.}

\textbf{Переналадка (перестановка ножей)} : изменение конфигурации ножей на ПРС при переходе между раскроями; приводит к простою оборудования и учитывается в модели через функцию $\sigma(i,j)$ и критерий $\tau(\overline{\overline{Y}})$.

\textbf{План раскроя (Cutting Plan)} : последовательность раскроев для одного ПРС; каждый раскрой содержит набор форматов и соответствующие мастер‑рулоны.

\textbf{План производства (Making Plan)} : последовательность тамбуров, которые должны быть произведены на одной БДМ.

\textbf{Плотность (граммаж)} : масса бумажного полотна на единицу площади (г/м²); влияет на скорость производства и смену параметров на БДМ.

\textbf{Полный перебор (Full Enumeration)} : алгоритм, перебирающий все допустимые комбинации раскроев и производственных планов; применим только для задач малой размерности.

\textbf{Популяция} : набор глобальных планов на каждой итерации генетического алгоритма; размер популяции влияет на качество решений и время расчёта.

\fixme{\textbf{Прогон имитации} : однократное воспроизведение исполнения расписания при заданном сценарии и фиксированном состоянии генератора случайных чисел; результатом является кортеж $\Omega$ с фактическим расписанием и показателями времени завершения и запаздываний заказов.}

\textbf{ПРС (продольно‑резательный станок)} : станок, который разрезает бумажный тамбур на конечные рулоны согласно плану раскроя; задаётся максимальной шириной обрабатываемого тамбура, количеством ножей, скоростью нарезки и временем перестановки ножей.

\textbf{Раскрой (раскройный план)} : организация нарезки мастер‑рулона на конечные рулоны; определяется набором форматов (ширин) и последовательностью их расположения на станке.

\textbf{Расписание} : упорядоченный список операций, определяющий расписание работы оборудования; обеспечивает согласование операций производства и раскроя во времени.

\textbf{Склад (склады рулонов)} : хранилище для тамбуров и рулонов, ограниченное по вместимости и допустимой ширине; используется для временного хранения между производством и раскроем.

\textbf{Промежуточный мастер‑рулон для~БРС} : рулон, получаемый в~результате нарезки тамбура на~ПРС и~предназначенный для~дальнейшей нарезки на~БРС; в~модели описывается форматом с~${g^Y}_{i,j,k}=0$, в~программной реализации представлен классом $\mathrm{SlittedMasterRoll}$.

\fixme{\textbf{Сценарий имитационного эксперимента} : набор параметров $\Theta$, задающий относительные отклонения длительностей операций, интенсивности отказов, среднее время ремонта, число прогонов и начальное состояние генератора случайных чисел; базовый сценарий (без возмущений) воспроизводит номинальное расписание.}

\textbf{Тамбур} : барабанный рулон бумаги, получаемый на БДМ; служит полуфабрикатом для последующей продольной резки.

\fixme{\textbf{Устойчивость плана} : способность глобального плана сохранять сроки выполнения заказов и приемлемое время завершения при случайных отклонениях длительностей операций и отказах оборудования; оценивается по статистическим показателям серии имитационных прогонов.}

\textbf{Формат (рулон)} : элемент раскроя, характеризующийся шириной $r^Y_{i,j,k}$ и номером заказа $g^Y_{i,j,k}$; если номер заказа равен~0, то формат предназначен для дальнейшей нарезки на БРС.

\textbf{Функция приспособленности} : значение целевой функции $\mu(G)$ для конкретного плана; используется для сравнения и отбора решений в генетическом алгоритме.

\textbf{Целевая функция} : векторная функция $\mu(G) = \langle \omega(G),\, \varphi(G),\, \tau(\overline{\overline{Y}}) \rangle \to \min$, компоненты которой отражают суммарные отходы, максимальное число раскроев на одной ПРС и общее количество перестановок ножей; минимизируется лексикографически.

\fixme{\textbf{Цех (производственное помещение)} : элемент множества~$S$, определяющий размещение оборудования; согласно цеховому ограничению передача тамбуров и рулонов между БДМ, складами и ПРС/БРС возможна только в пределах одного помещения.}
